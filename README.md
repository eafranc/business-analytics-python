# Business Analytics with Python

Exploratory data analysis projects simulating the kind of business insights
typically delivered through SAP Analytics Cloud (SAC) and SAP Datasphere.

Each notebook analyzes a real-world business dataset, covering data cleaning,
aggregation, visualization, and interpretation — bridging the gap between
raw data and actionable reporting.
